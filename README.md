# vertex-organizer
A minimalist productivity app combining task management with note-taking. Built with HTML, CSS, and JavaScript.

🚀 Features:

📋 To-Do List:

-Add tasks with title, description, due dates, and priority levels

-Mark tasks as complete with visual indicators

-Priority-based organization (High, Medium, Low with color coding)

-Filter tasks by status (All, Pending, Completed)

-Task management with edit and delete functionality

-Persistent storage using browser's local storage


📝 Note-Taking:

-Create rich notes with titles and detailed content

-Tag system for organizing notes by category

-Visual note cards with clean, responsive design

-Quick actions for managing notes

-Search-friendly organization


🎨 User Interface:

-Modern, clean design with intuitive navigation

-Responsive layout that works on desktop and mobile

-Visual statistics dashboard showing task completion progress

-Real-time updates without page refresh

-Empty state illustrations for better UX
